# Contributing to VibeNav

Thanks for your interest! Here's how to help:

## Getting Started
1. Fork the repo
2. Open in Android Studio
3. Connect your Android device via WiFi ADB or USB

## Development
- All Kotlin source in `app/src/main/java/com/vibenav/`
- Uses osmdroid + OpenStreetMap (no API keys needed)
- Min SDK 26, target SDK 34

## Pull Requests
- Keep changes focused — one feature per PR
- Test on a real Samsung/Android device
- Update `versionCode` in `app/build.gradle.kts` for releases
- Match existing code style (no comments, concise)

## Reporting Issues
- **Bugs**: Open an issue with device model, Android version, and steps to reproduce
- **Features**: Open a discussion first to gauge interest
